# 나만의 이모티콘 생성 서비스 - 브레인스토밍 요약

> 날짜: 2026-03-04
> 상태: 구체화중
> 수정: 카카오톡 → 네이버 OGQ 중심으로 피봇 (카카오 AI 이모티콘 입점 제한 정책 반영)

## 아이디어 개요

내 얼굴 사진 1장을 업로드하면 AI가 다양한 표정/포즈의 이모티콘 세트(24~32개)를 자동 생성해주는 서비스. 네이버 OGQ마켓 규격으로 출력하여 바로 판매할 수 있고, 생성한 이모티콘을 자체 마켓플레이스에서도 판매할 수 있는 복합 플랫폼.

## 타겟 사용자

| 세그먼트 | 설명 | 니즈 |
|---------|------|------|
| MZ세대 일반 사용자 | 카톡/인스타에서 나만의 이모티콘을 쓰고 싶은 10~30대 | 쉬운 생성, 재미, 공유 |
| 이모티콘 작가 지망생 | 이모티콘 제작을 배우고 싶지만 그림 실력이 부족한 사람 | AI 보조 제작, OGQ/카카오 제출 |
| 기업/브랜드 | 브랜드 캐릭터 이모티콘을 만들고 싶은 소상공인/기업 | 빠른 브랜드 이모티콘 생성 |

## 해결하려는 문제

1. **이모티콘 제작 진입장벽**: 그림 실력이 없으면 이모티콘을 만들 수 없다
2. **플랫폼 규격 번거로움**: OGQ/카카오톡 등 각 플랫폼별 규격 맞추기가 복잡하다
3. **개인화 부재**: 기존 이모티콘은 내 얼굴/모습이 아닌 범용 캐릭터
4. **시간/비용**: 외주 제작 시 50만원~, 직접 제작 시 수주~수개월 소요

## 핵심 기능

1. **사진 → 이모티콘 자동 생성**: 셀카 1장 업로드 → AI가 24~32개 표정/포즈 세트 생성
2. **스타일 선택**: 카툰, 플랫, 애니메, 만화, 3D, 수채화 등 다양한 스타일
3. **네이버 OGQ 마켓 규격 자동 출력**: 스티커/애니메이션 스티커 규격으로 자동 변환 + UCI 발급 연동
4. **텍스트/말풍선 추가**: 이모티콘에 한글 텍스트, 말풍선 자유 입력

## 차별점 / 강점

- **네이버 OGQ 마켓 연동**: 생성 → 규격 자동 맞춤 → OGQ 크리에이터 스튜디오 제출까지 원스톱
  - OGQ는 AI 생성 콘텐츠를 **공식 허용**하고 UCI(국가콘텐츠식별체계) 발급까지 제공
- **한국형 표정/문화 특화**: 한국 메신저 문화에 맞는 표정 세트 (아이고, 케이크, 화내는, 눈물 등)
- **AI 품질 (닮음도)**: 원본 얼굴과 높은 유사도 유지하면서 다양한 표정 생성
- **작가 등록/판매 마켓플레이스**: 생성한 이모티콘을 플랫폼 내에서 다른 사용자에게 판매 가능
- **카카오톡 대비 준비**: 카카오 AI 정책 변경 시 즉시 대응 가능한 규격 맞춤 기능 사전 구현

## 서비스 형태

- **웹 앱 (SaaS)**: 브라우저에서 사진 업로드 → 이모티콘 다운로드 (메인)
- **모바일 앱**: iOS/Android 앱으로 카메라 셀카 → 바로 생성 (확장)
- **네이버 밴드/카톡 개인 사용**: 생성된 이모티콘을 개인 스티커로 바로 사용 (확장)

**핵심 발견**: 네이버 OGQ 규격에 특화된 얼굴 기반 AI 이모티콘 생성 서비스는 **아직 없다**. 큰 기회.

## 기술적 고려사항

- **AI 이미지 생성**: SDXL + IP-Adapter-FaceID (1순위), fal.ai API (MVP용)
- **얼굴 인식**: InsightFace/ArcFace로 얼굴 임베딩 추출
- **표정 제어**: ControlNet (OpenPose/MediaPipe)으로 24~32개 표정 프리셋
- **백그라운드 제거**: rembg / SAM2로 투명 배경 자동 처리
- **이미지 후처리**: 플랫폼별 규격 리사이징, PNG 최적화
- **서버 인프라**: RunPod Serverless (L40S), ~$0.05/세트
- **프론트엔드**: Next.js + React (웹), React Native (모바일)
